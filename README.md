# 🚆 Railway Reservation System (C++)

## 📌 Project Description

Railway Reservation System is a simple C++ project developed using Object-Oriented Programming (OOP) concepts. The main purpose of this project is to manage train records in an easy and organized way through a menu-driven program.

The system allows users to add new train records, display all available train records, search for a train using its train number, and exit the program. This project is useful for understanding the implementation of OOP concepts such as Classes, Objects, Constructors, Destructors, Static Members, Getter & Setter Functions, Encapsulation, Arrays of Objects, Loops, and Switch Case.

This project is designed for beginners who want to learn C++ programming and understand how real-life systems can be developed using Object-Oriented Programming.

## 🚀 How to Run

### Step 1

Open the project in **Visual Studio Code**.

### Step 2

Compile the program using:

```bash
g++ railway-system.cpp
```

### Step 3

Run the program:

```bash
./a.exe
```

### Step 4

Choose any option from the menu.

* Add New Train Record
* Display All Train Records
* Search Train by Train Number
* Exit
